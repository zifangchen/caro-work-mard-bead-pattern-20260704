# Caro Work-Mode MARD Bead Pattern

This folder contains the deliverables for the Caro work-mode bead pattern:

- `caro-work-mard-bead-pattern.png`: labeled MARD bead chart, 58 x 70 beads
- `caro-work-mard-pixel-preview.png`: pixel-only preview
- `caro-work-frame.png`: extracted source frame
- `caro-work-mard-color-counts.csv`: bead color counts
- `caro-work-mard-package.zip`: all files packaged together

Palette source: https://www.pixel-beads.com/zh/mard-bead-color-chart
